# RollThaDice

A simple dice roller in Python

## Usage:
python roll.py [x]d[y] [OPTIONS]

### Options:

-n [numbers]     roll n times the dice(s)

-h              Show this help message

## Example:

python roll.py 3d6

Roll'd da Dice

[4, 6, 5]

python roll.py 1d20 -n 5

Roll'd da Dices

[10]

[20]

[7]

[15]

[15]
